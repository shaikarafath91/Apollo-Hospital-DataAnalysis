# Apollo-Hospital-DataAnalysis
Hospital Workflow & Revenue Analysis Dashboard
📌 Project Overview

This project focuses on analyzing hospital operational and billing data using an end-to-end data analytics workflow.

The objective of this project is to solve key business questions related to:

Diagnosis trends

Doctor-wise revenue generation

Patient length of stay

Insurance coverage gap

Bed occupancy utilization

Overall hospital performance

The project follows an industry-style approach:
Excel → MySQL → Power BI

🛠 Tools & Technologies Used

MS Excel – Data Cleaning & Preprocessing

MySQL – Data Storage & SQL-Based Analysis

Power BI – Dashboard Development & Data Visualization

📊 Project Workflow
1️⃣ Data Cleaning (Excel)

Removed duplicate records

Handled missing values

Standardized date format to YYYY-MM-DD

Ensured compatibility with MySQL DATE datatype

Prepared structured dataset for database import

2️⃣ Data Analysis (MySQL)

Imported the cleaned dataset into MySQL and solved important business questions using SQL queries such as:

What are the most common diagnosis categories?

Which doctor generates the highest total billing revenue?

What is the length of stay for each patient?

Who are the top 5 highest billing patients?

Which medical test is most frequently performed?

What is the actual amount paid by patients after insurance coverage?

SQL Concepts Used:

SELECT

WHERE

GROUP BY

SUM()

COUNT()

DATEDIFF()

ORDER BY

LIMIT

3️⃣ Dashboard Development (Power BI)

Built an interactive dashboard with:

KPI Cards (Total Billing Revenue, Average Feedback)

Diagnosis-wise distribution chart

Bed occupancy breakdown (ICU, Private, General)

Billing vs Insurance comparison

Doctor-wise performance analysis

Patient ID and Date Range slicers

📈 Key Insights Generated

Identified high-frequency diagnosis categories

Determined highest revenue-generating doctors

Analyzed patient stay duration trends

Measured insurance contribution vs billing revenue

Evaluated hospital bed utilization efficiency

🎯 Business Impact

This dashboard helps hospital management:

Monitor revenue performance

Analyze operational efficiency

Understand patient distribution trends

Identify insurance coverage gaps

Support data-driven decision making
